# Get-AzComputeResourceSku

## SYNOPSIS
모든 계산 자원 Sku 나열

## 구문과

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
모든 계산 자원 Sku 나열

## 예제의

### 예제 1
```
PS C:\> Get-AzComputeResourceSku "westus";
```

미국 지역에 모든 계산 자원 sku 나열
