# Invoke-AzureRemoteAppSessionLogoff

## SYNOPSIS
Azure RemoteApp 세션을 즉시 로그 오프 합니다.

## 구문과

```
Invoke-AzureRemoteAppSessionLogoff [-CollectionName] <String> [-UserUpn] <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRemoteAppSessionLogoff** Cmdlet은 Azure RemoteApp에서 실행 되는 원격 세션에서 사용자를 즉시 로그 오프 합니다.

## 예제의

### 예제 1: 사용자 로그 오프
```
PS C:\> Invoke-AzureRemoteAppSessionLogoff -CollectionName ContosoApps -UserUpn PattiFuller@contoso.com
```

이 명령은 UPN이 있는 사용자를 로그 오프 PattiFuller@contoso.com 합니다.
