# Disable-AzureRmHDInsightOperationsManagementSuite

## SYNOPSIS
HDInsight 클러스터에서 OMS (Operations Management Suite)를 사용 하지 않도록 설정 하면 관련 로그가 사용 중에 지정 된 OMS 작업 영역으로 이동 하지 않습니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Disable-AzureRmHDInsightOperationsManagementSuite [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmHDInsightOperationsManagementSuite** Cmdlet은 Azure HDInsight 클러스터에서 OMS (Operations Management Suite)를 사용 하지 않도록 설정 합니다.

## 예제의

### 예제 1
```
PS C:\> Disable-AzureRmHDInsightOMS -Name testcluster

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

HDInsight 클러스터에서 OMS (Operations Management Suite)가 사용 되지 않도록 설정 되며 관련 로그가 OMS 작업 영역으로 이동 하지 않습니다.

### 예제 2
```
PS C:\> Disable-AzureRmHDInsightOMS -Name testcluster -ResourceGroupName testrg

ErrorInfo  :

```

HDInsight 클러스터에서 OMS (Operations Management Suite)가 사용 되지 않도록 설정 되며 관련 로그가 OMS 작업 영역으로 이동 하지 않습니다.
