# Az DataBox 모듈
## 설명은
Databox 작업을 만들고, 취소 하 고, 삭제 하 고, 가져오는 모듈

## Az DataBox Cmdlet
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
특정 작업의 databox 자격 증명을 가져옵니다.

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Databox 작업에 대 한 정보를 가져옵니다.

### [새로운 AzDataBoxJob](New-AzDataBoxJob.md)
지정 된 매개 변수를 사용 하 여 새 databox 작업을 만듭니다.

### [제거-AzDataBoxJob](Remove-AzDataBoxJob.md)
Databox 작업을 삭제 합니다.

### [중지-AzDataBoxJob](Stop-AzDataBoxJob.md)
작업이 cancellable 상태인 경우 진행 중인 databox 작업을 취소 합니다.
