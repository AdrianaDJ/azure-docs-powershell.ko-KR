# New-AzP2sVpnGateway

## SYNOPSIS
사이트 연결을 위한 VirtualHub 아래에 새 P2SVpnGateway를 만듭니다.

## 설명은
**AzP2sVpnGateway** cmdlet을 사용 하면 사이트 클라이언트에서 Azure virtualhub으로의 위치에 대 한 끝점에 대해 virtualhub에서 새 P2SVpnGateway를 만들 수 있습니다.
