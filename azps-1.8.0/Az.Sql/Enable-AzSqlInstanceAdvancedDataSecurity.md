# Enable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
관리 되는 인스턴스에서 고급 데이터 보안을 사용 하도록 설정 합니다.

## 구문과

```
Enable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명은
AzSqlInstanceAdvancedDataSecurity cmdlet을 **사용** 하면 관리 되는 인스턴스에서 고급 데이터 보안을 사용할 수 있습니다.

## 예제의

### 예제 1-관리 되는 인스턴스 사용 고급 데이터 보안
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### 예제 2-서버 리소스의 관리 되는 인스턴스 고급 데이터 보안 사용
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
