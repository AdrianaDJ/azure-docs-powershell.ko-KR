# AzureRM 모듈
## 설명은
이 섹션의 항목에서는 azure Resource Manager (ARM) 프레임 워크의 Azure CDN (콘텐츠 배달 네트워크) 용 Azure PowerShell cmdlet을 설명 합니다. 이 cmdlet은 Microsoft Azure. Cdn 네임 스페이스에 있습니다.
