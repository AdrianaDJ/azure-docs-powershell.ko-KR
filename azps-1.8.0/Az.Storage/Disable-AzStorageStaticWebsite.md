# Disable-AzStorageStaticWebsite

## SYNOPSIS
Azure 저장소 계정에 대 한 정적 웹 사이트를 사용 하지 않도록 설정 합니다.

## 구문과

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzStorageStaticWebsite** Cmdlet은 Azure 저장소 계정에 대 한 정적 웹 사이트를 사용 하지 않도록 설정 합니다.

## 예제의

### 예제 1: Azure 저장소 계정에 정적 웹 사이트 사용 안 함
```
C:\PS>Disable-AzStorageStaticWebsite
```

이 명령은 Azure 저장소 계정에 대 한 정적 웹 사이트를 사용 하지 않도록 설정 합니다.
