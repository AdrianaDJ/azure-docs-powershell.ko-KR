# Get-AzureRmApiManagementIdentityProvider

## SYNOPSIS
Id 공급자 구성 세부 정보를 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

### AllIdentityProviders (기본값)
```
Get-AzureRmApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProviderByType
```
Get-AzureRmApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
Id 공급자 구성 세부 정보를 가져옵니다.

## 예제의

### --------------------------예제 1--------------------------
@ {paragraph = PS C: \\ \> }







```
Get-AzureRmApiManagementIdentityProvider -Context $context
```

서비스에서 모든 id 공급자 구성을 가져옵니다.

### --------------------------예제 2--------------------------
@ {paragraph = PS C: \\ \> }







```
Get-AzureRmApiManagementIdentityProvider -Context $context -Type Aad
```

Azure Active Directory의 Id 공급자 구성을 가져옵니다.
