## SYNOPSIS
가상 컴퓨터의 백업 확장 속성을 설정 합니다.

## 구문과

```
Set-AzVMBackupExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Tag] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```
