# Get-AzSqlServer

## SYNOPSIS
SQL 데이터베이스 서버에 대 한 정보를 반환 합니다.

## 구문과

```
Get-AzSqlServer [[-ResourceGroupName] <String>] [[-ServerName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzSqlServer** cmdlet은 하나 이상의 Azure SQL 데이터베이스 서버에 대 한 정보를 반환 합니다.
해당 서버에 대 한 정보를 볼 서버의 이름을 지정 합니다.

## 예제의

### 예제 1: 리소스 그룹에 할당 된 SQL Server의 모든 인스턴스 가져오기
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net

ResourceGroupName        : resourcegroup01
ServerName               : server02
Location                 : West US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server02.database.windows.net
```

이 명령은 리소스 그룹 ResourceGroup01에 할당 된 모든 Azure SQL 데이터베이스 서버에 대 한 정보를 가져옵니다.

### 예제 2: Azure SQL 데이터베이스 서버에 대 한 정보 가져오기
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net
```

이 명령은 Server01 이라는 Azure SQL 데이터베이스 서버에 대 한 정보를 가져옵니다.

### 예제 3: 구독에서 SQL Server의 모든 인스턴스 가져오기
```

Location                 : Central US

Identity                 :

ResourceGroupName        : resourcegroup01
ServerName               : server02
Location                 : West US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server02.database.windows.net

ResourceGroupName        : resourcegroup02
ServerName               : server03
Location                 : East US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server03.database.windows.net
```

이 명령은 현재 구독의 모든 Azure SQL 데이터베이스 서버에 대 한 정보를 가져옵니다.

### 예제 4: 필터링을 사용 하 여 리소스 그룹에 할당 된 SQL Server의 모든 인스턴스 가져오기
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "server*"
```

이 명령은 "server"로 시작 하는 리소스 그룹 ResourceGroup01에 할당 된 모든 Azure SQL 데이터베이스 서버에 대 한 정보를 가져옵니다.
