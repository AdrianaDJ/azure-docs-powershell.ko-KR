# Az OperationalInsights 모듈
## 설명은
이 항목에서는 Azure Operational Insights Cmdlet에 대 한 도움말 항목을 표시 합니다.

## Az OperationalInsights Cmdlet
### [Disable-AzOperationalInsightsIISLogCollection](Disable-AzOperationalInsightsIISLogCollection.md)
컴퓨터에서 IIS 로그 수집을 중지 합니다.

### [Disable-AzOperationalInsightsLinuxCustomLogCollection](Disable-AzOperationalInsightsLinuxCustomLogCollection.md)
Linux 컴퓨터에서 사용자 지정 로그 수집을 중지 합니다.

### [Disable-AzOperationalInsightsLinuxPerformanceCollection](Disable-AzOperationalInsightsLinuxPerformanceCollection.md)
Linux 컴퓨터에서 성능 카운터 수집을 중지 합니다.

### [Disable-AzOperationalInsightsLinuxSyslogCollection](Disable-AzOperationalInsightsLinuxSyslogCollection.md)
Linux 컴퓨터에서 syslog 데이터 수집을 중지 합니다.

### [Enable-AzOperationalInsightsIISLogCollection](Enable-AzOperationalInsightsIISLogCollection.md)
작업 영역의 컴퓨터에서 IIS 로그 수집을 시작 합니다.

### [Enable-AzOperationalInsightsLinuxCustomLogCollection](Enable-AzOperationalInsightsLinuxCustomLogCollection.md)
Linux 컴퓨터에서 사용자 지정 로그 수집을 시작 합니다.

### [Enable-AzOperationalInsightsLinuxPerformanceCollection](Enable-AzOperationalInsightsLinuxPerformanceCollection.md)
Linux 컴퓨터에서 성능 카운터 모음을 시작 합니다.

### [Enable-AzOperationalInsightsLinuxSyslogCollection](Enable-AzOperationalInsightsLinuxSyslogCollection.md)
Linux 컴퓨터에서 syslog 데이터 수집을 시작 합니다.

### [Get-AzOperationalInsightsDataSource](Get-AzOperationalInsightsDataSource.md)
Azure Log Analytics 작업 영역에서 데이터 원본을 가져옵니다.

### [Get-AzOperationalInsightsIntelligencePack](Get-AzOperationalInsightsIntelligencePack.md)
사용할 수 있는 인텔리전스 팩을 가져옵니다.

### [Get-AzOperationalInsightsLinkTarget](Get-AzOperationalInsightsLinkTarget.md)
구독과 연결 되지 않은 계정을 가져옵니다.

### [Get-AzOperationalInsightsSavedSearch](Get-AzOperationalInsightsSavedSearch.md)
지정 된 작업 영역에 대해 저장 된 모든 검색을 반환 합니다.

### [Get-AzOperationalInsightsSavedSearchResult](Get-AzOperationalInsightsSavedSearchResult.md)
쿼리 결과를 반환 합니다.

### [Get-AzOperationalInsightsSchema](Get-AzOperationalInsightsSchema.md)
작업 영역과 연결 된 스키마를 반환 합니다.

### [Get-AzOperationalInsightsSearchResult](Get-AzOperationalInsightsSearchResult.md)
지정 된 매개 변수를 기준으로 검색 결과를 반환 합니다.

### [Get-AzOperationalInsightsStorageInsight](Get-AzOperationalInsightsStorageInsight.md)
저장소 통찰에 대 한 정보를 가져옵니다.

### [Get-AzOperationalInsightsWorkspace](Get-AzOperationalInsightsWorkspace.md)
작업 영역에 대 한 정보를 가져옵니다.

### [Get-AzOperationalInsightsWorkspaceManagementGroup](Get-AzOperationalInsightsWorkspaceManagementGroup.md)
작업 영역에 연결 된 관리 그룹의 세부 정보를 가져옵니다.

### [Get-AzOperationalInsightsWorkspaceSharedKey](Get-AzOperationalInsightsWorkspaceSharedKey.md)
작업 영역의 공유 키를 가져옵니다.

### [Get-AzOperationalInsightsWorkspaceUsage](Get-AzOperationalInsightsWorkspaceUsage.md)
작업 영역에 대 한 사용 현황 데이터를 가져옵니다.

### [AzOperationalInsightsQuery-호출](Invoke-AzOperationalInsightsQuery.md)
지정 된 매개 변수를 기준으로 검색 결과를 반환 합니다.

### [새로운 AzOperationalInsightsApplicationInsightsDataSource](New-AzOperationalInsightsApplicationInsightsDataSource.md)
지정 된 Application-Insights 응용 프로그램에서 로그를 수집 합니다.

### [새로운 AzOperationalInsightsAzureActivityLogDataSource](New-AzOperationalInsightsAzureActivityLogDataSource.md)
지정 된 구독에서 Azure Activity 로그를 수집 합니다.

### [새로운 AzOperationalInsightsComputerGroup](New-AzOperationalInsightsComputerGroup.md)
컴퓨터 그룹을 만듭니다.

### [새로운 AzOperationalInsightsCustomLogDataSource](New-AzOperationalInsightsCustomLogDataSource.md)
사용자 지정 로그 수집 정책을 정의 합니다.

### [새로운 AzOperationalInsightsLinuxPerformanceObjectDataSource](New-AzOperationalInsightsLinuxPerformanceObjectDataSource.md)
작업 영역의 모든 Linux 컴퓨터에 성능 카운터를 추가 합니다.

### [새로운 AzOperationalInsightsLinuxSyslogDataSource](New-AzOperationalInsightsLinuxSyslogDataSource.md)
Linux 컴퓨터에 데이터 원본을 추가 합니다.

### [새로운 AzOperationalInsightsSavedSearch](New-AzOperationalInsightsSavedSearch.md)
지정 된 매개 변수를 사용 하 여 새 저장 된 검색을 만듭니다.

### [새로운 AzOperationalInsightsStorageInsight](New-AzOperationalInsightsStorageInsight.md)
작업 영역 내에서 저장소를 파악 합니다.

### [새로운 AzOperationalInsightsWindowsEventDataSource](New-AzOperationalInsightsWindowsEventDataSource.md)
Windows 운영 체제를 실행 하는 컴퓨터에서 이벤트 로그를 수집 합니다.

### [새로운 AzOperationalInsightsWindowsPerformanceCounterDataSource](New-AzOperationalInsightsWindowsPerformanceCounterDataSource.md)
Windows 운영 체제를 실행 하는 연결 된 컴퓨터에 대 한 Windows 성능 카운터 데이터 원본을 추가 합니다.

### [새로운 AzOperationalInsightsWorkspace](New-AzOperationalInsightsWorkspace.md)
작업 영역을 만듭니다.

### [제거-AzOperationalInsightsDataSource](Remove-AzOperationalInsightsDataSource.md)
데이터 원본을 삭제 합니다.

### [제거-AzOperationalInsightsSavedSearch](Remove-AzOperationalInsightsSavedSearch.md)
작업 영역에서 저장 된 검색을 제거 합니다.

### [제거-AzOperationalInsightsStorageInsight](Remove-AzOperationalInsightsStorageInsight.md)
저장소 통찰력을 제거 합니다.

### [제거-AzOperationalInsightsWorkspace](Remove-AzOperationalInsightsWorkspace.md)
작업 영역을 제거 합니다.

### [Set-AzOperationalInsightsDataSource](Set-AzOperationalInsightsDataSource.md)
데이터 원본을 업데이트 합니다.

### [Set-AzOperationalInsightsIntelligencePack](Set-AzOperationalInsightsIntelligencePack.md)
지정 된 인텔리전스 팩을 사용 하거나 사용 하지 않도록 설정 합니다.

### [Set-AzOperationalInsightsSavedSearch](Set-AzOperationalInsightsSavedSearch.md)
이미 존재 하는 저장 된 검색을 업데이트 합니다.

### [Set-AzOperationalInsightsStorageInsight](Set-AzOperationalInsightsStorageInsight.md)
저장소 통찰력을 업데이트 합니다.

### [Set-AzOperationalInsightsWorkspace](Set-AzOperationalInsightsWorkspace.md)
작업 영역을 업데이트 합니다.
