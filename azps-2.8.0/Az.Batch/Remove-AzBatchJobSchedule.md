# Remove-AzBatchJobSchedule

## SYNOPSIS
일괄 작업 일정을 제거 합니다.

## 구문과

```
Remove-AzBatchJobSchedule [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzBatchJobSchedule** Cmdlet은 Azure Batch 작업 일정을 제거 합니다.

## 예제의

### 예제 1: 일괄 작업 일정 삭제
```
PS C:\>Remove-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context
```

이 명령은 ID MyJobSchedule 인 작업 일정을 삭제 합니다.
이 명령은 작업을 삭제 하기 전에 확인을 요청 하 라는 메시지를 표시 합니다.
Get-AzBatchAccountKeys cmdlet을 사용 하 여 컨텍스트를 $Context 변수에 할당 합니다.

### 예제 2: 파이프라인을 사용 하 여 확인 없이 일괄 작업 삭제
```
PS C:\>Get-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context | Remove-AzBatchJobSchedule -Force -BatchContext $Context
```

이 명령은 Get-AzBatchJobSchedule cmdlet을 사용 하 여 ID MyJobSchedule 인 작업 일정을 가져옵니다.
이 명령은 파이프라인 연산자를 사용 하 여 해당 작업 일정을 현재 cmdlet에 전달 합니다.
이 명령은 해당 작업 일정을 삭제 합니다.
명령에 *Force* 매개 변수가 포함 되어 있기 때문에 확인을 요청 하는 메시지가 표시 되지 않습니다.
