# Remove-AzureRmRedisCache

## SYNOPSIS
Redis 캐시를 제거 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Remove-AzureRmRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmRedisCache** Cmdlet은 Azure Redis Cache를 제거 합니다.

## 예제의

### 예제 1: Redis Cache를 제거 하 고 결과 반환
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

이 명령은 Redis 캐시를 제거 하 고 작업의 성공 여부를 표시 합니다.

### 예제 2: Redis Cache 제거 및 결과 표시 안 함
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

이 명령은 Redis 캐시를 제거 합니다.
*PassThru* 매개 변수를 지정 하지 않았으므로 연산 결과가 표시 되지 않습니다.
