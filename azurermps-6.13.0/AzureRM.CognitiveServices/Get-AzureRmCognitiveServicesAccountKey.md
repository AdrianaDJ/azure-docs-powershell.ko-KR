# Get-AzureRmCognitiveServicesAccountKey

## SYNOPSIS
계정에 대 한 API 키를 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmCognitiveServicesAccountKey** cmdlet은 프로 비전 된 인식 서비스 계정에 대 한 API 키를 가져옵니다.
인지 서비스 계정에는 Key1 및 Key2의 두 가지 API 키가 있습니다.
키를 사용 하면 인식 서비스 계정 끝점과 상호 작용할 수 있습니다.
New-AzureRmCognitiveServicesAccountKey를 사용 하 여 키를 다시 생성 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Get-AzureRmCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
