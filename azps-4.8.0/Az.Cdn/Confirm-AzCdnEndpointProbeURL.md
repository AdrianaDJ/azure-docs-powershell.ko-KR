# Confirm-AzCdnEndpointProbeURL

## SYNOPSIS
프로브 URL의 유효성을 검사 합니다.

## 구문과

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
AzCdnEndpointProbeURL cmdlet은 제공 된 프로브 URL을 동적 사이트 가속에 사용할 수 있는지 **확인** 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

검색 url ""의 유효성을 검사 합니다. http://www.bing.com/images
