# Set-AzCdnOrigin

## SYNOPSIS
CDN 원본 서버를 업데이트 합니다.

## 구문과

```
Set-AzCdnOrigin -CdnOrigin <PSOrigin> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzCdnOrigin** Cmdlet은 Azure CDN (콘텐츠 배달 네트워크) 원본 서버를 업데이트 합니다.
