# Enable-AzSqlServerAdvancedDataSecurity

## SYNOPSIS
서버에서 고급 데이터 보안을 사용 하도록 설정 합니다.

## 구문과

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명은
AzSqlServerAdvancedDataSecurity cmdlet을 **사용** 하면 서버에서 고급 데이터 보안을 사용할 수 있습니다. 고급 데이터 보안은 서버에 대 한 데이터 분류, 취약점 평가 및 고급 위협 보호를 포함 하는 통합 된 보안 패키지입니다. 취약점 평가를 저장 하기 위해 새로운 저장소 계정이 자동으로 생성 됩니다. 이전에이 용도로 만든 저장소 계정이 아닌 경우 사용 됩니다.

## 예제의

### 예제 1-서버 고급 데이터 보안 사용
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### 예제 2-서버 리소스에서 서버 고급 데이터 보안 설정
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
