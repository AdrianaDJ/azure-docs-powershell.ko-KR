# Suspend-AzAnalysisServicesServer

## SYNOPSIS
Analysis Services 서버 인스턴스 일시 중단

## 구문과

```
Suspend-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
Suspend-AzAnalysisServicesServer cmdlet이 Analysis Services 서버 인스턴스를 일시 중단 합니다.

## 예제의

### 예제 1
```
PS C:\> Suspend-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

이 명령은 resourcegroup testserver에서 testserver 라는 활성 서버를 일시 중단 합니다.
