# Get-AzEventHubCluster

## SYNOPSIS
단일 이벤트 허브 클러스터의 세부 정보를 가져오거나 이벤트 허브 클러스터의 목록을 가져옵니다.

## 구문과

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
Get-AzEventHubCluster cmdlet은 이벤트 허브 클러스터의 세부 정보 또는 주어진 리소스 그룹에 있는 모든 이벤트 허브 클러스터의 목록을 반환 합니다.
클러스터 이름을 제공 하는 경우 단일 클러스터에 대 한 세부 정보가 반환 됩니다.
클러스터 이름을 제공 하지 않으면 지정 된 리소스 그룹에 있는 모든 클러스터의 목록이 반환 됩니다.
