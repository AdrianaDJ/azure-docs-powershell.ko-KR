# Get-AzsOffer

## SYNOPSIS
루트 공급자에 대 한 공개 서비스 목록을 가져옵니다.

## 구문과

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 설명은
루트 공급자에 대 한 공개 서비스 목록을 가져옵니다.
