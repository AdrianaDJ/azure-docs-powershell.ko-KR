# Test-AzureRmDataLakeAnalyticsAccount

## SYNOPSIS
Data Lake Analytics 계정이 있는지 확인 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Test-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmDataLakeAnalyticsAccount** Cmdlet은 Data Lake Analytics 계정이 있는지 확인 합니다.

## 예제의

### 예제 1: 계정이 있는지 테스트
```
PS C:\>Test-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

이 명령은 ContosoAdlAccount 이라는 계정이 있는지 여부를 테스트 합니다.
