# New-AzFirewallPolicyNetworkRule

## SYNOPSIS
새 Azure 방화벽 정책 네트워크 규칙 만들기

## 구문과

```
New-AzFirewallPolicyNetworkRule -Name <String> [-Description <String>] -SourceAddress <String[]>
 [-SourceIpGroup <String[]>] -DestinationAddress <String[]> [-DestinationIpGroup <String[]>]
 -DestinationPort <String[]> [-DestinationFqdn <String[]>] -Protocols <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzFirewallPolicyNetworkRule** Cmdlet은 Azure 방화벽 정책에 대 한 네트워크 규칙을 만듭니다.

## 예제의

### 예제 1
```powershell
PS C:\> New-AzFirewallPolicyNetworkRule -Name NRC1 -Protocol "TCP" -SourceAddress "192.168.0.0/16" -DestinationAddress * -DestinationPort *
```

이 예제에서는 원본 주소, 프로토콜, 대상 주소 및 대상 포트를 사용 하 여 네트워크 규칙을 만듭니다.
