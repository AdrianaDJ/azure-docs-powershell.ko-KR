# New-AzureRouteTable

## SYNOPSIS
경로 테이블을 만듭니다.

## 구문과

```
New-AzureRouteTable -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## 설명은
**AzureRouteTable** cmdlet은 지정 된 위치에 경로 테이블을 만듭니다.
경로 테이블에 사용자 정의 경로를 추가할 수 있습니다.
가상 네트워크의 서브넷에 경로 테이블을 연결할 수 있습니다.
