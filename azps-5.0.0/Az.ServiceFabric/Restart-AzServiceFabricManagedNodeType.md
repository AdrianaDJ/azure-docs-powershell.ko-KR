# Restart-AzServiceFabricManagedNodeType

## SYNOPSIS
노드 형식에서 특정 노드를 다시 시작 합니다.

## 구문과

```
Restart-AzServiceFabricManagedNodeType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 -NodeName <String[]> [-ForceRestart] [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명은
노드 형식에서 특정 노드를 다시 시작 합니다. Vm을 다시 시작 하기 전에 service fabric 노드가 비활성화 되 고 다시 도착 하면 다시 다시 활성화 됩니다. 기본 노드 형식에서이 작업을 수행 하는 경우에는 동시에 모든 노드를 다시 시작 하지 않을 수 있으므로 시간이 걸릴 수 있습니다. Use-ForceRestart는 서비스 패브릭이 노드를 사용 하지 않도록 설정할 수 없지만,이로 인해 노드에서 상태 저장 작업이 실행 되는 경우 데이터가 손실 될 수 있으므로 주의 해 서 사용 합니다.

## 예제의

### 예제 1
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Restart-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName  -Name $NodeTypeName -NodeName nt1_0, nt1_3
```

노드 형식에서 노드 0 및 3을 다시 시작 합니다.
