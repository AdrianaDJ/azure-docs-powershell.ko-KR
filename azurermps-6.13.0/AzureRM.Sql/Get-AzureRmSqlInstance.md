# Get-AzureRmSqlInstance

## SYNOPSIS
Azure SQL 관리 데이터베이스 인스턴스에 대 한 정보를 반환 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

### GetInstanceByResourceGroup (기본값)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**Get-AzureRmSqlInstance** cmdlet은 하나 이상의 Azure SQL 관리 인스턴스에 대 한 정보를 반환 합니다.
해당 인스턴스에 대 한 정보만 보려면 인스턴스의 이름을 지정 합니다.
