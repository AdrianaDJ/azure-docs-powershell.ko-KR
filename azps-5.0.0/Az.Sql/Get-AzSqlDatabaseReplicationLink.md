# Get-AzSqlDatabaseReplicationLink

## SYNOPSIS
Azure SQL 데이터베이스와 리소스 그룹 또는 SQL Server 간의 지역/복제 링크를 가져옵니다.

## 구문과

```
Get-AzSqlDatabaseReplicationLink [-DatabaseName] <String> -PartnerResourceGroupName <String>
 [-PartnerServerName <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzSqlDatabaseReplicationLink** Cmdlet은 **AzSqlDatabaseCopy** cmdlet을 대신 합니다.
지정 된 Azure SQL 데이터베이스와 리소스 그룹 또는 AzureSQL 서버 간의 모든 지역/복제 링크를 가져옵니다.

## 예제의

### 예제 1

Azure SQL 데이터베이스와 리소스 그룹 또는 SQL Server 간의 지역/복제 링크를 가져옵니다. 생성
