# New-AzApplicationGatewayFirewallExclusionConfig

## SYNOPSIS
Application gateway waf에 대 한 새 제외 규칙 목록 만들기

## 구문과

```
New-AzApplicationGatewayFirewallExclusionConfig -Variable <String> -Operator <String> -Selector <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzApplicationGatewayFirewallExclusionConfig** cmdlet은 application gateway waf에 대 한 새 제외 규칙 목록입니다.

## 예제의

### 예제 1
```powershell
PS C:\> $exclusion1 = New-AzApplicationGatewayFirewallExclusionConfig -Variable "RequestHeaderNames" -Operator "StartsWith" -Selector "xyz"
```

이 명령은 새 제외 규칙을 만들고 xyz 라는 RequestHeaderNames 및 StartsWith 이라는 변수에 대 한 구성을 나열 합니다. 제외 목록 구성은 $exclusion 1에 저장 됩니다.
