# Remove-AzRouteTable

## SYNOPSIS
경로 테이블을 제거 합니다.

## 구문과

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzRouteTable** Cmdlet은 Azure 경로 테이블을 제거 합니다.

## 예제의

### 예제 1: 경로 테이블 제거
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

이 명령은 ResourceGroup11 이라는 리소스 그룹에서 RouteTable01 이라는 경로 테이블을 제거 합니다.
테이블을 제거 하기 전에 확인 하 라는 메시지가 cmdlet에 표시 됩니다.
