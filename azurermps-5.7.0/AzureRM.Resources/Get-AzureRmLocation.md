# Get-AzureRmLocation

## SYNOPSIS
각 위치에 대해 지원 되는 리소스 공급자 및 모든 위치를 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmLocation [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
**AzureRmLocation** cmdlet은 각 위치에 대해 지원 되는 리소스 공급자와 모든 위치를 가져옵니다.

### 예제 1: 모든 위치 및 지원 되는 리소스 공급자 가져오기
```
PS C:\>Get-AzureRmLocation
```
