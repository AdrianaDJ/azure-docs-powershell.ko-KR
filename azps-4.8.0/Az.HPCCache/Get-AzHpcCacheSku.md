# Get-AzHpcCacheSku

## SYNOPSIS
구독에서 사용할 수 있는 모든 Sku를 가져옵니다.

## 구문과

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzHpcCacheSku** cmdlet은 구독에서 사용할 수 있는 sku 목록을 반환 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
