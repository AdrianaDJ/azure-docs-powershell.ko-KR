# Stop-AzureRmSqlServerUpgrade

## SYNOPSIS
SQL 데이터베이스 서버의 업그레이드를 중지 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmSqlServerUpgrade** Cmdlet은 Azure SQL 데이터베이스 서버의 업그레이드를 중지 합니다.

## 예제의

### 예제 1: 서버 업그레이드 중지
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

이 명령은 ResourceGroup01 이라는 리소스 그룹에 할당 된 Server02 서버를 업그레이드 하는 요청을 중지 합니다.
