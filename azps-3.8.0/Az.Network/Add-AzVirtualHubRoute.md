# Add-AzVirtualHubRoute

## SYNOPSIS
Add-AzVirtualHubRouteTable 명령에 매개 변수로 전달 될 수 있는 VirtualHubRoute 개체를 만듭니다. 

## 구문과

```
Add-AzVirtualHubRoute -Destination <String[]> -DestinationType <String> -NextHop <String[]>
 -NextHopType <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
VirtualHubRoute 개체를 만듭니다.

## 예제의

### 예제 1

```powershell
NextHopType      : IPAddress
NextHops         : {10.0.0.68}
```

위 명령에서는 VirtualHubRouteTable 리소스에 추가 하 고 VirtualHub로 설정할 수 있는 VirtualHubRoute 개체를 만듭니다.
