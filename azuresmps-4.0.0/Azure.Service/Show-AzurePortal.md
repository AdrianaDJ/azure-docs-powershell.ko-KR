# Show-AzurePortal

## SYNOPSIS
Azure 관리 포털을 표시 합니다.

## 구문과

```
Show-AzurePortal [-Name <String>] [-Realm <String>] [-Environment <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## 설명은
이 항목에서는 0.8.10 버전의 Microsoft Azure PowerShell 모듈에 대 한 cmdlet에 대해 설명 합니다.
사용 중인 모듈 버전을 Azure PowerShell 콘솔에 가져오려면 다음을 입력 `(Get-Module -Name Azure).Version` 합니다.

**AzurePortal** Cmdlet은 Azure 관리 포털을 표시 합니다.

## 예제의

### 예제 1: 웹 사이트에 대 한 정보 표시
```
PS C:\> Show-AzurePortal -Name mySite
```

이 예제에서는 Azure 포털에서 브라우저가 열려 있는 웹 사이트에 대 한 정보를 보여 줍니다.
