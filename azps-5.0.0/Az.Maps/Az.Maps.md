# Az 모듈
## 설명은
이 항목에서는 Azure Maps Cmdlet에 대 한 도움말을 표시 합니다.

## Az Cmdlet
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
계정을 가져옵니다.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
계정에 대 한 API 키를 가져옵니다.
이러한 키는 Azure Maps에 대 한 후속 호출에 사용 되는 인증 메커니즘입니다.

### [새로운 AzMapsAccount](New-AzMapsAccount.md)
Azure Maps 계정을 만듭니다.

### [새로운 AzMapsAccountKey](New-AzMapsAccountKey.md)
계정 키를 다시 생성 합니다.

### [제거-AzMapsAccount](Remove-AzMapsAccount.md)
Azure Maps 계정을 삭제 합니다.
