# New-AzureRmApplicationSecurityGroup

## SYNOPSIS
응용 프로그램 보안 그룹을 만듭니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
New-AzureRmApplicationSecurityGroup -ResourceGroupName <String> -Name <String> -Location <String>
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명은
**AzureRmApplicationSecurityGroup** cmdlet은 응용 프로그램 보안 그룹을 만듭니다.

## 예제의

### 예제 1
```
PS C:\> New-AzureRmApplicationSecurityGroup -ResourceGroupName "MyResourceGroup" -Name "MyApplicationSecurityGroup" -Location "West US"
```

이 예제에서는 연결이 없는 응용 프로그램 보안 그룹을 만듭니다. 생성 된 네트워크 인터페이스의 IP 구성은 그룹에 포함 될 수 있습니다. 또한 보안 규칙은 그룹을 원본 또는 대상으로 참조할 수 있습니다.
