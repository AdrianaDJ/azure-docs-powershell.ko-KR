# Az. IotCentral 모듈
## 설명은
IoT 중앙 응용 프로그램에 대 한 CRUD 관리를 제공 합니다.

## Az. IotCentral Cmdlet
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
하나 또는 여러 IoT 중앙 응용 프로그램에 대 한 속성을 가져옵니다.

### [새로운 AzIotCentralApp](New-AzIotCentralApp.md)
새 IoT 중앙 응용 프로그램을 만듭니다.

### [제거-AzIotCentralApp](Remove-AzIotCentralApp.md)
IoT 중앙 응용 프로그램을 삭제 합니다.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
IoT 중앙 응용 프로그램에 대 한 메타 데이터를 업데이트 합니다.
