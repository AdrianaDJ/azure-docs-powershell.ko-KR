# Use-AzureRmHDInsightCluster

## SYNOPSIS
Invoke-RmAzureHDInsightHiveJob cmdlet에 사용할 클러스터를 선택 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Use-AzureRmHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmHDInsightCluster** Cmdlet은 Hive 작업을 제출 하는 데 사용할 Invoke-AzureRmHDInsightHiveJob cmdlet에 대 한 Azure HDInsight 클러스터를 선택 합니다.

## 예제의

### 예제 1: 하이브 쿼리 제출을 위한 클러스터 선택
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzureRmHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

이 명령은 하이브 쿼리 제출에 대 한 클러스터를 선택 합니다.
