# Test-AzureRmDataLakeStoreAccount

## SYNOPSIS
Data Lake Store 계정이 있는지 테스트 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**테스트 AzureRmDataLakeStoreAccount** Cmdlet은 Data Lake store 계정이 있는지 테스트 합니다.

## 예제의

### 예제 1: 계정 테스트
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

이 명령은 ContosoADL 이라는 계정이 있는지 여부를 테스트 합니다.
