# Start-AzureRmSiteRecoveryTestFailoverJob

## SYNOPSIS
사이트 복구 보호 엔터티에 대 한 테스트 장애 조치를 시작 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```

 [-DataEncryptionPrimaryCertFile <String>] [-DataEncryptionSecondaryCertFile <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmSiteRecoveryTestFailoverJob** Cmdlet은 Azure Site Recovery 보호 엔터티 또는 복구 계획의 장애 조치 테스트를 시작 합니다.
Get-AzureRmSiteRecoveryJob cmdlet을 사용 하 여 작업 성공 여부를 확인할 수 있습니다.
