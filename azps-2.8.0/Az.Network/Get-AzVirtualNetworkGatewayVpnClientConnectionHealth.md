# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## SYNOPSIS 
Vpn 클라이언트 연결당 Azure 가상 네트워크 게이트웨이의 vpn 클라이언트 연결 상태 목록 가져오기

## 구문과

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## 설명은
연결 된 모든 vpn 클라이언트 연결 상태를 나열 합니다. 여기에는 vpnConnectionId vpnConnectionDuration vpnConnectionTime publicIpAddress privateIpAddress vpnUserName maxBandwidth egressPacketsTransferred egressBytesTransferred ingressPacketsTransferred ingressBytesTransferred maxPacketsPerSecond이 포함 됩니다.
