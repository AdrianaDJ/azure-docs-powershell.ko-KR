# Az 공간 모듈
## 설명은
Azure DevSpaces 컨트롤러와 상호 작용 하는 명령입니다.

## Az 공간 Cmdlet
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Azure DevSpaces 컨트롤러를 가져오거나 나열 합니다.

### [새로운 AzDevSpacesController](New-AzDevSpacesController.md)
새 Azure DevSpaces 컨트롤러를 만듭니다.

### [제거-AzDevSpacesController](Remove-AzDevSpacesController.md)
DevSpaces 컨트롤러를 삭제 합니다.

### [업데이트-AzDevSpacesController](Update-AzDevSpacesController.md)
DevSpaces 컨트롤러를 업데이트 하 여 태그를 추가 합니다.
