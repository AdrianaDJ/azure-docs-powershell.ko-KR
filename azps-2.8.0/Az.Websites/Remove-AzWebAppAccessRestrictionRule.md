# Remove-AzWebAppAccessRestrictionRule

## SYNOPSIS
Azure 웹 앱에서 액세스 제한 규칙을 제거 합니다.

## 구문과

```
Remove-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> -Name <String>
 [-TargetScmSite] [-SlotName <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명은
AzWebAppAccessRestrictionRule cmdlet이 Azure 웹 앱에서 액세스 제한 규칙을 제거 **함**

## 예제의

### 예제 1: 웹 앱 액세스 제한 규칙 제거
```
PS C:\>Remove-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" -Name IpRule
```

이 명령은 Default-WestUS 이라는 리소스 그룹에 속하는 Azure Web App ContosoSite의 IpRule 액세스 제한 규칙을 제거 합니다.
