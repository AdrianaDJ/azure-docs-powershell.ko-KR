# Get-AzureRemoteAppPlan

## SYNOPSIS
Azure RemoteApp 계획을 나열 합니다.

## 구문과

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 설명은
**AzureRemoteAppPlan** Cmdlet은 Azure RemoteApp 계획을 나열 합니다.
