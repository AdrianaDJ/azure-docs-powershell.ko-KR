# Get-AzSnapshot

## SYNOPSIS
스냅숏의 속성을 가져옵니다.

## 구문과

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**Get-AzSnapshot** cmdlet은 스냅숏의 속성을 가져옵니다.
