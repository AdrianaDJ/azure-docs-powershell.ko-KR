# Remove-AzResourceGroupDeployment

## SYNOPSIS
리소스 그룹 배포 및 연결 된 작업을 제거 합니다.

## 구문과

### RemoveByResourceGroupName (기본값)
```
Remove-AzResourceGroupDeployment [-ResourceGroupName] <String> [-Name] <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupDeploymentId
```
Remove-AzResourceGroupDeployment -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은

**AzResourceGroupDeployment** Cmdlet은 Azure 리소스 그룹 배포 및 연결 된 작업을 제거 합니다.

## 예제의

### 예제 1: ResourceId를 사용 하 여 리소스 그룹 배포 제거

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceId /subscriptions/{subId}/resourceGroups/testGroup/providers/Microsoft.Resources/deployments/testDeployment1

True
```

이 명령은 배포의 정규화 된 리소스 Id를 사용 하 여 리소스 그룹 배포를 제거 합니다.
성공적으로 제거 되 면 true를 반환 합니다.

### 예제 2: ResourceGroupName 및 Context.resourcename을 사용 하 여 리소스 그룹 배포 제거

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceGroupName testGroup -Name testDeployment1

True
```

이 명령은 제공 된 ResourceGroupName 및 Context.resourcename을 사용 하 여 리소스 그룹 배포를 제거 합니다.
성공적으로 제거 되 면 true를 반환 합니다.
