# ManagedServiceIdentity 모듈 AzureRM
## 설명은
이 항목에서는 Azure 관리 되는 서비스 Id Cmdlet에 대 한 도움말 항목을 표시 합니다.

## ManagedServiceIdentity Cmdlet AzureRM
### [Get-AzureRmUserAssignedIdentity](Get-AzureRmUserAssignedIdentity.md)
사용자가 할당 한 Id/id를 가져옵니다.

### [새로운 AzureRmUserAssignedIdentity](New-AzureRmUserAssignedIdentity.md)
새 사용자에 게 할당 된 Id를 만들거나 기존 사용자에 게 할당 된 Id를 업데이트 합니다.

### [제거-AzureRmUserAssignedIdentity](Remove-AzureRmUserAssignedIdentity.md)
사용자가 할당 한 Id를 제거 합니다.
