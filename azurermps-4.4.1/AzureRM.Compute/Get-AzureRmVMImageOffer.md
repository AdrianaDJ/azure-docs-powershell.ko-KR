# Get-AzureRmVMImageOffer

## SYNOPSIS
VMImage 제공 형식을 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
**AzureRmVMImageOffer** Cmdlet은 VMImage 제안 형식을 가져옵니다.

## 예제의

### 예제 1: 게시자에 대 한 제안 형식 가져오기
```
PS C:\> Get-AzureRmVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

이 명령은 중앙 US 지역에서 지정 된 게시자에 대 한 제공 유형을 가져옵니다.
