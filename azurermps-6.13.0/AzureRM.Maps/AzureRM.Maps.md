# AzureRM 모듈
## 설명은
이 항목에서는 Azure Maps Cmdlet에 대 한 도움말을 표시 합니다.

## AzureRM Cmdlet
### [Get-AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
계정을 가져옵니다.

### [Get-AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
계정에 대 한 API 키를 가져옵니다.
이러한 키는 Azure Maps에 대 한 후속 호출에 사용 되는 인증 메커니즘입니다.

### [새로운 AzureRmMapsAccount](New-AzureRmMapsAccount.md)
Azure Maps 계정을 만듭니다.

### [새로운 AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
계정 키를 다시 생성 합니다.

### [제거-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
Azure Maps 계정을 삭제 합니다.
