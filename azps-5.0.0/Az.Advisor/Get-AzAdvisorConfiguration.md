# Get-AzAdvisorConfiguration

## SYNOPSIS
지정 된 구독 또는 리소스 그룹에 대 한 Azure Advisor 구성을 가져옵니다.

## 구문과

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
구독과 연결 된 구성에는 두 가지 유형이 있습니다.

구독 수준 구성: 구독에 대해이 유형의 구성은 하나만 있을 수 있습니다. LowCpuThreshold 및 Exclude는이 유형의 구성에 대 한 유일한 속성입니다.

ResourceGroup 수준 구성: 구독의 각 ResourceGroup에 대 한 구성은 하나만 있을 수 있습니다. Exclude는이 유형의 구성에 대 한 유일한 속성입니다.
