# Register-AzModule

## SYNOPSIS
내부용 전용-AutoRest 생성 된 cmdlet에 대 한 런타임 지원을 제공 합니다.

## 구문과

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
내부용 전용-AutoRest 생성 된 cmdlet에 대 한 런타임 지원을 제공 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Register-AzModule
```

AutoRest 생성 된 cmdlet에서 내부적으로 사용 됩니다.
