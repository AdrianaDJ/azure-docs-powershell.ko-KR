# Get-AzDtlAutoShutdownPolicy

## SYNOPSIS
DevTest 랩에서 랩에서 자동 종료 정책을 가져옵니다.

## 구문과

```
Get-AzDtlAutoShutdownPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzDtlAutoShutdownPolicy** cmdlet은 랩의 자동 종료 정책을 가져와 지정 된 시간에 랩에서 모든 가상 컴퓨터를 자동으로 종료할 수 있습니다.
Cmdlet은 정책의 상태를 사용할지 여부와 랩 가상 컴퓨터를 자동으로 종료 하도록 설정한 시간을 반환 합니다.
