DISKTOINCLUDE IVMwareCbtDiskInput [] >: 복제에 포함할 원본 서버의 디스크를 지정 합니다.
  - `DiskId <String>`: 디스크 Id입니다.
  - `IsOSDisk <String>`: 디스크가 OS 디스크 인지 여부를 나타내는 값입니다.
  - `LogStorageAccountId <String>`: 로그 저장소 계정 팔 Id입니다.
  - `LogStorageAccountSasSecretName <String>`: 로그 저장소 계정의 키 자격 증명 정보 비밀 이름입니다.
  - `[DiskEncryptionSetId <String>]`: Diska 암 번호를 설정 합니다.
  - `[DiskType <DiskAccountType?>]`: 디스크 유형입니다.

INPUTOBJECT <IVMwareMachine> : 마이그레이션할 검색 된 서버를 지정 합니다. 서버 개체는 Get-AzMigrateServer cmdlet을 사용 하 여 검색할 수 있습니다.
