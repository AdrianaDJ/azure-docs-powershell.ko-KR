# Set-AzRouteTable

## SYNOPSIS
경로 테이블의 목표 상태를 설정 합니다.

## 구문과

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명은
**Set-AzRouteTable** Cmdlet은 Azure 경로 테이블에 대 한 목표 상태를 설정 합니다.
