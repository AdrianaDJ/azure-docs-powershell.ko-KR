# Remove-AzureRouteTable

## SYNOPSIS
경로 테이블을 제거 합니다.

## 구문과

```
Remove-AzureRouteTable -Name <String> [-PassThru] [-Force] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 설명은
**AzureRouteTable** cmdlet은 구독에서 경로 테이블을 제거 합니다.
경로 테이블이 서브넷에 연결 된 경우에는 테이블을 제거할 수 없습니다.

## 예제의

### 예제 1: 경로 테이블 제거
```
PS C:\> Remove-AzureRouteTable -Name "PublicRouteTable"
```

이 명령은 PublicRouteTable 이라는 경로 테이블을 제거 합니다.
