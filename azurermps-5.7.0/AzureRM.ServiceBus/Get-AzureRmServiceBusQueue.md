# Get-AzureRmServiceBusQueue

## SYNOPSIS
지정 된 Service Bus 큐에 대 한 설명을 반환 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmServiceBusQueue** cmdlet은 지정 된 Service Bus 큐에 대 한 설명을 반환 합니다.
