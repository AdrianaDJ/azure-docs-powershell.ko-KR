# Set-AzureRmDefault

## SYNOPSIS
현재 컨텍스트에서 기본값을 설정 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Set-AzureRmDefault [-ResourceGroupName <String>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 설명은
Set-AzureRmDefault cmdlet은 현재 컨텍스트의 기본값을 추가 하거나 변경 합니다.

## 예제의

### 예제 1
```
PS C:\> Set-AzureRmDefault -ResourceGroupName myResourceGroup

Name       : myResourceGroup

```

이 명령은 기본 리소스 그룹을 사용자가 지정한 리소스 그룹으로 설정 합니다.
