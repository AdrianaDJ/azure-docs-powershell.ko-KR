# Remove-AzureRmServiceEndpointPolicy

## SYNOPSIS
{{Synopsis 채우기}}

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Remove-AzureRmServiceEndpointPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmServiceEndpointPolicy** cmdlet은 서비스 끝점 정책을 제거 합니다.

## 예제의

### 예제 1: 이름을 사용 하 여 서비스 끝점 정책 제거
```
Remove-AzureRmServiceEndpointPolicy -Name "Policy1" -ResourceGroup "resourcegroup1"
```

이 명령은 이름이 "resourcegroup1" 인 resourcegroup에 속하는 이름 Policy1 인 서비스 끝점 정책을 제거 합니다.

### 예제 2: 입력 개체를 사용 하 여 서비스 끝점 정책 제거
```
Remove-AzureRmServiceEndpointPolicy -InputObject $Policy1 -ResourceGroup "resourcegroup1"
```

이 명령은 이름이 "resourcegroup1" 인 resourcegroup에 속하는 서비스 끝점 정책 개체 Policy1를 제거 합니다.
