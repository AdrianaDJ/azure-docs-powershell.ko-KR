# Remove-AzureVMBGInfoExtension

## SYNOPSIS
가상 컴퓨터에 적용 된 BGInfo 확장을 제거 합니다.

## 구문과

```
Remove-AzureVMBGInfoExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## 설명은
**AzureVMBGInfoExtension** cmdlet은 가상 컴퓨터에 적용 된 BGInfo 확장을 제거 합니다.

## 예제의

### 예제 1: 가상 컴퓨터에서 BGInfo 확장 제거
```
PS C:\> Remove-AzureVMBGInfoExtension -VM $VM;
```

이 명령은 가상 컴퓨터에 적용 된 BGInfo 확장을 제거 합니다.
