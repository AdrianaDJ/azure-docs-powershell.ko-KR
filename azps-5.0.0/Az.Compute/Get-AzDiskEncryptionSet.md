# Get-AzDiskEncryptionSet

## SYNOPSIS
디스크 암호화 집합을 가져오거나 나열 합니다.

## 구문과

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
디스크 암호화 집합을 가져오거나 나열 합니다.
