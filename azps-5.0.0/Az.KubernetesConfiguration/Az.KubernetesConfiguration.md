# Az KubernetesConfiguration 모듈
## 설명은
Microsoft Azure PowerShell: KubernetesConfiguration cmdlet

## Az KubernetesConfiguration Cmdlet
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
소스 제어 구성에 대 한 세부 정보를 가져옵니다.

### [새로운 AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
새 Kubernetes 원본 컨트롤 구성을 만듭니다.

### [제거-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
이렇게 하면 소스 제어 구성을 설정 하는 데 사용 되는 YAML 파일이 삭제 되므로 이후 원본 리포지토리에서의 동기화가 중지 됩니다.
