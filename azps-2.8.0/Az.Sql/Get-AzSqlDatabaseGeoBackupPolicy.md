# Get-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
데이터베이스 지역 백업 정책을 가져옵니다.

## 구문과

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzSqlDatabaseGeoBackupPolicy** cmdlet은이 데이터베이스에 등록 된 지역 백업 정책을 가져옵니다.
백업 저장소 정책을 정의 하는 데 사용 되는 Azure Backup 리소스입니다.
