# Get-AzSqlSyncAgent

## SYNOPSIS
Azure SQL 동기화 에이전트에 대 한 정보를 반환 합니다.

## 구문과

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzSqlSyncAgent** cmdlet은 하나 이상의 Azure SQL 동기화 에이전트에 대 한 정보를 반환 합니다.
동기화 에이전트의 이름을 지정 하 여 해당 동기화 에이전트에 대 한 정보만 표시 합니다.
