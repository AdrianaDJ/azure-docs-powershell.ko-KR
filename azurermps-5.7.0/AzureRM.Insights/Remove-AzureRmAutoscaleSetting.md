# Remove-AzureRmAutoscaleSetting

## SYNOPSIS
자동 크기 조정 설정을 제거 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Remove-AzureRmAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmAutoscaleSetting** Cmdlet은 자동 크기 조정 설정을 제거 합니다.
설정의 이름과이를 할당 받은 리소스 그룹의 이름을 지정 해야 합니다.

이 cmdlet은 ShouldProcess 패턴을 구현 합니다. 즉, 리소스를 실제로 생성, 수정 또는 제거 하기 전에 사용자에 게 확인을 요청할 수 있습니다.
