# StorageSync 모듈 AzureRm
## 설명은
저장소 동기화 모듈의 cmdlet을 사용 하면 PowerShell의 Azure 파일 동기화와 관련 된 작업을 관리할 수 있습니다.

## StorageSync Cmdlet AzureRm
### [AzureRmStorageSyncCompatibilityCheck-호출](Invoke-AzureRmStorageSyncCompatibilityCheck.md)
시스템 및 Azure 파일 동기화 간에 발생할 수 있는 호환성 문제가 있는지 확인 합니다.
