# OperationalInsights 모듈 AzureRM
## 설명은
이 항목에서는 Azure Operational Insights Cmdlet에 대 한 도움말 항목을 표시 합니다.

## OperationalInsights Cmdlet AzureRM
### [Disable-AzureRmOperationalInsightsIISLogCollection](Disable-AzureRmOperationalInsightsIISLogCollection.md)
컴퓨터에서 IIS 로그 수집을 중지 합니다.

### [Disable-AzureRmOperationalInsightsLinuxCustomLogCollection](Disable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
Linux 컴퓨터에서 사용자 지정 로그 수집을 중지 합니다.

### [Disable-AzureRmOperationalInsightsLinuxPerformanceCollection](Disable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
Linux 컴퓨터에서 성능 카운터 수집을 중지 합니다.

### [Disable-AzureRmOperationalInsightsLinuxSyslogCollection](Disable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
Linux 컴퓨터에서 syslog 데이터 수집을 중지 합니다.

### [Enable-AzureRmOperationalInsightsIISLogCollection](Enable-AzureRmOperationalInsightsIISLogCollection.md)
작업 영역의 컴퓨터에서 IIS 로그 수집을 시작 합니다.

### [Enable-AzureRmOperationalInsightsLinuxCustomLogCollection](Enable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
Linux 컴퓨터에서 사용자 지정 로그 수집을 시작 합니다.

### [Enable-AzureRmOperationalInsightsLinuxPerformanceCollection](Enable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
Linux 컴퓨터에서 성능 카운터 모음을 시작 합니다.

### [Enable-AzureRmOperationalInsightsLinuxSyslogCollection](Enable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
Linux 컴퓨터에서 syslog 데이터 수집을 시작 합니다.

### [Get-AzureRmOperationalInsightsDataSource](Get-AzureRmOperationalInsightsDataSource.md)
Azure Log Analytics 작업 영역에서 데이터 원본을 가져옵니다.

### [Get-AzureRmOperationalInsightsIntelligencePacks](Get-AzureRmOperationalInsightsIntelligencePacks.md)
사용할 수 있는 인텔리전스 팩을 가져옵니다.

### [Get-AzureRmOperationalInsightsLinkTargets](Get-AzureRmOperationalInsightsLinkTargets.md)
구독과 연결 되지 않은 계정을 가져옵니다.

### [Get-AzureRmOperationalInsightsSavedSearch](Get-AzureRmOperationalInsightsSavedSearch.md)
지정 된 작업 영역에 대해 저장 된 모든 검색을 반환 합니다.

### [Get-AzureRmOperationalInsightsSavedSearchResults](Get-AzureRmOperationalInsightsSavedSearchResults.md)
쿼리 결과를 반환 합니다.

### [Get-AzureRmOperationalInsightsSchema](Get-AzureRmOperationalInsightsSchema.md)
작업 영역과 연결 된 스키마를 반환 합니다.

### [Get-AzureRmOperationalInsightsSearchResults](Get-AzureRmOperationalInsightsSearchResults.md)
지정 된 매개 변수를 기준으로 검색 결과를 반환 합니다.

### [Get-AzureRmOperationalInsightsStorageInsight](Get-AzureRmOperationalInsightsStorageInsight.md)
저장소 통찰에 대 한 정보를 가져옵니다.

### [Get-AzureRmOperationalInsightsWorkspace](Get-AzureRmOperationalInsightsWorkspace.md)
작업 영역에 대 한 정보를 가져옵니다.

### [Get-AzureRmOperationalInsightsWorkspaceManagementGroups](Get-AzureRmOperationalInsightsWorkspaceManagementGroups.md)
작업 영역에 연결 된 관리 그룹의 세부 정보를 가져옵니다.

### [Get-AzureRmOperationalInsightsWorkspaceSharedKeys](Get-AzureRmOperationalInsightsWorkspaceSharedKeys.md)
작업 영역의 공유 키를 가져옵니다.

### [Get-AzureRmOperationalInsightsWorkspaceUsage](Get-AzureRmOperationalInsightsWorkspaceUsage.md)
작업 영역에 대 한 사용 현황 데이터를 가져옵니다.

### [새로운 AzureRmOperationalInsightsAzureActivityLogDataSource](New-AzureRmOperationalInsightsAzureActivityLogDataSource.md)
지정 된 구독에서 Azure Activity 로그를 수집 합니다.

### [새로운 AzureRmOperationalInsightsComputerGroup](New-AzureRmOperationalInsightsComputerGroup.md)
컴퓨터 그룹을 만듭니다.

### [새로운 AzureRmOperationalInsightsCustomLogDataSource](New-AzureRmOperationalInsightsCustomLogDataSource.md)
사용자 지정 로그 수집 정책을 정의 합니다.

### [새로운 AzureRmOperationalInsightsLinuxPerformanceObjectDataSource](New-AzureRmOperationalInsightsLinuxPerformanceObjectDataSource.md)
작업 영역의 모든 Linux 컴퓨터에 성능 카운터를 추가 합니다.

### [새로운 AzureRmOperationalInsightsLinuxSyslogDataSource](New-AzureRmOperationalInsightsLinuxSyslogDataSource.md)
Linux 컴퓨터에 데이터 원본을 추가 합니다.

### [새로운 AzureRmOperationalInsightsSavedSearch](New-AzureRmOperationalInsightsSavedSearch.md)
지정 된 매개 변수를 사용 하 여 새 저장 된 검색을 만듭니다.

### [새로운 AzureRmOperationalInsightsStorageInsight](New-AzureRmOperationalInsightsStorageInsight.md)
작업 영역 내에서 저장소를 파악 합니다.

### [새로운 AzureRmOperationalInsightsWindowsEventDataSource](New-AzureRmOperationalInsightsWindowsEventDataSource.md)
Windows 운영 체제를 실행 하는 컴퓨터에서 이벤트 로그를 수집 합니다.

### [새로운 AzureRmOperationalInsightsWindowsPerformanceCounterDataSource](New-AzureRmOperationalInsightsWindowsPerformanceCounterDataSource.md)
Windows 운영 체제를 실행 하는 연결 된 컴퓨터에 대 한 Windows 성능 카운터 데이터 원본을 추가 합니다.

### [새로운 AzureRmOperationalInsightsWorkspace](New-AzureRmOperationalInsightsWorkspace.md)
작업 영역을 만듭니다.

### [제거-AzureRmOperationalInsightsDataSource](Remove-AzureRmOperationalInsightsDataSource.md)
데이터 원본을 삭제 합니다.

### [제거-AzureRmOperationalInsightsSavedSearch](Remove-AzureRmOperationalInsightsSavedSearch.md)
작업 영역에서 저장 된 검색을 제거 합니다.

### [제거-AzureRmOperationalInsightsStorageInsight](Remove-AzureRmOperationalInsightsStorageInsight.md)
저장소 통찰력을 제거 합니다.

### [제거-AzureRmOperationalInsightsWorkspace](Remove-AzureRmOperationalInsightsWorkspace.md)
작업 영역을 제거 합니다.

### [Set-AzureRmOperationalInsightsDataSource](Set-AzureRmOperationalInsightsDataSource.md)
데이터 원본을 업데이트 합니다.

### [Set-AzureRmOperationalInsightsIntelligencePack](Set-AzureRmOperationalInsightsIntelligencePack.md)
지정 된 인텔리전스 팩을 사용 하거나 사용 하지 않도록 설정 합니다.

### [Set-AzureRmOperationalInsightsSavedSearch](Set-AzureRmOperationalInsightsSavedSearch.md)
이미 존재 하는 저장 된 검색을 업데이트 합니다.

### [Set-AzureRmOperationalInsightsStorageInsight](Set-AzureRmOperationalInsightsStorageInsight.md)
저장소 통찰력을 업데이트 합니다.

### [Set-AzureRmOperationalInsightsWorkspace](Set-AzureRmOperationalInsightsWorkspace.md)
작업 영역을 업데이트 합니다.
