# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
관리 되는 인스턴스에서 고급 데이터 보안을 사용 하지 않도록 설정 합니다.

## 구문과

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명은
**AzSqlInstanceAdvancedDataSecurity** cmdlet은 관리 되는 인스턴스에서 고급 데이터 보안을 사용 하지 않도록 설정 합니다.

## 예제의

### 예제 1-관리 되는 인스턴스 사용 안 함 고급 데이터 보안
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### 예제 2-관리 되는 인스턴스 리소스에서 서버 고급 데이터 보안을 사용 하지 않도록 설정
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
