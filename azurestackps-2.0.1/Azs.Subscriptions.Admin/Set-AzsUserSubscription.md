# Set-AzsUserSubscription

## SYNOPSIS
지정 된 구독을 만들거나 업데이트 합니다.

## 구문과

### UpdateExpanded 됨 (기본값)
```
Set-AzsUserSubscription -TargetSubscriptionId <String> [-SubscriptionId <String>]
 [-DelegatedProviderSubscriptionId <String>] [-DisplayName <String>] [-ExternalReferenceId <String>]
 [-Id <String>] [-OfferId <String>] [-Owner <String>] [-RoutingResourceManagerType <ResourceManagerType>]
 [-State <SubscriptionState>] [-SubscriptionId1 <String>] [-TenantId <String>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Update
```
Set-AzsUserSubscription -SubscriptionDefinition <ISubscriptionDefinition> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## 설명은
지정 된 구독을 만들거나 업데이트 합니다.

PS C:\> $Subscription = Get-AzsUserSubscription | Select-Object -First 1

## 상속자

복잡 한 매개 변수 속성 아래 설명 된 매개 변수를 만들려면 적절 한 속성을 포함 하는 해시 테이블을 생성 합니다. 해시 테이블에 대 한 자세한 내용은 Get-Help about_Hash_Tables를 실행 하세요.

SUBSCRIPTIONDEFINITION <ISubscriptionDefinition> : 구독 개체 속성입니다.
  - `[DelegatedProviderSubscriptionId <String>]`: Parent DelegatedProvider 구독 식별자입니다.
  - `[DisplayName <String>]`: 구독 이름입니다.
  - `[ExternalReferenceId <String>]`: 외부 참조 식별자입니다.
  - `[Id <String>]`: 정규화 된 식별자입니다.
  - `[OfferId <String>]`: 위임 된 공급자 범위 아래에 있는 제안의 식별자입니다.
  - `[Owner <String>]`: 구독 소유자입니다.
  - `[RoutingResourceManagerType <ResourceManagerType?>]`: 라우팅 리소스 관리자 유형입니다.
  - `[State <SubscriptionState?>]`: 구독 상태입니다.
  - `[SubscriptionId <String>]`: 구독 식별자입니다.
  - `[TenantId <String>]`: 디렉터리 테 넌 트 식별자입니다.
