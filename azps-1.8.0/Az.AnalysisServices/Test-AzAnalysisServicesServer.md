# Test-AzAnalysisServicesServer

## SYNOPSIS
Analysis Services 서버의 인스턴스가 있는지 테스트 합니다.

## 구문과

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
Test-AzAnalysisServicesServer cmdlet은 Analysis Services 서버 인스턴스가 있는지 테스트 합니다.

## 예제의

### 예제 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

이 명령은 resourcegroup testserver에 testserver 라는 서버가 있는지 테스트 합니다.
