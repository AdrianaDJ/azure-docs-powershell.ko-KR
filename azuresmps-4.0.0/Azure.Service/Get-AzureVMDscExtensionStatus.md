# Get-AzureVMDscExtensionStatus

## SYNOPSIS
클라우드 서비스 또는 서비스의 특정 가상 컴퓨터에 배포 된 모든 가상 컴퓨터에 대 한 DSC 확장 상태를 가져옵니다.

## 구문과

### GetStatusByServiceAndVMName (기본값)
```
Get-AzureVMDscExtensionStatus [-ServiceName] <String> [[-Name] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

### GetStatusByVM
```
Get-AzureVMDscExtensionStatus -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## 설명은
**AzureVMDscExtensionStatus** cmdlet은 클라우드 서비스에 배포 된 모든 가상 컴퓨터 또는 서비스의 특정 가상 컴퓨터에 대 한 DSC (원하는 상태 구성) 확장 상태를 가져옵니다.
