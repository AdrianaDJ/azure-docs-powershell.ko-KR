# Azs-Keyvault. 관리 모듈
## 설명은
관리자가 KeyVault 할당량을 볼 수 있도록 하는 AzureStack KeyVault 관리자 모듈의 미리 보기 릴리스. 

## Azs. Keyvault. 관리 Cmdlet
### [Get-AzsKeyVaultQuota](Get-AzsKeyVaultQuota.md)
위치에 있는 KeyVault에 대 한 모든 할당량 개체의 목록을 가져옵니다.
