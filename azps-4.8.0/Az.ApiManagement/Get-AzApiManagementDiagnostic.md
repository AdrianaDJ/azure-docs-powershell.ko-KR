# Get-AzApiManagementDiagnostic

## SYNOPSIS
서비스 수준 또는 Api 수준에서 구성 된 진단에 대 한 세부 정보를 가져옵니다. 진단은 Api Management 게이트웨이에서 요청/응답을 기록 하는 데 사용 됩니다.

## 구문과

### ContextParameterSet (기본값)
```
Get-AzApiManagementDiagnostic -Context <PsApiManagementContext> [-DiagnosticId <String>] [-ApiId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementDiagnostic -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
**AzApiManagementDiagnostic** 는 지정 된 범위에서 Api management 서비스에 구성 된 진단 정보를 가져옵니다.

PS C:\>Get-AzApiManagementDiagnostic -Context $apimContext

ServiceName                  : contoso

DiagnosticId                 : azuremonitor
ApiId                        :

EnableHttpCorrelationHeaders :

이 명령은 Api Management 서비스에 구성 된 모든 진단을 가져옵니다.
