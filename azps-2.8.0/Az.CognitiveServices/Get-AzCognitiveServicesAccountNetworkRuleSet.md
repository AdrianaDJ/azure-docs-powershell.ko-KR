# Get-AzCognitiveServicesAccountNetworkRuleSet

## SYNOPSIS
인지 서비스 계정의 NetworkRule 속성 가져오기

## 구문과

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzCognitiveServicesAccountNetworkRuleSet** Cmdlet은 인지 서비스 계정의 networkrule 속성을 가져옵니다.

## 예제의

### 예제 1: 지정 된 인식 서비스 계정의 NetworkRule 속성 가져오기
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

이 명령은 지정 된 인식 서비스 계정의 NetworkRule 속성을 가져옵니다.
