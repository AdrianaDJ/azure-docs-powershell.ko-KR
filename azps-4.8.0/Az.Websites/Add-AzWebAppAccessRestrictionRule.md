# Add-AzWebAppAccessRestrictionRule

## SYNOPSIS
Azure Web App에 Access Restiction 규칙을 추가 합니다.

## 구문과

### IpAddressParameterSet (기본값)
```
Add-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> [-Name <String>]
 [-Description <String>] -Priority <UInt32> [-Action <String>] [-SlotName <String>] [-TargetScmSite]
 -IpAddress <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SubnetNameParameterSet
```
Add-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> [-Name <String>]

 -SubnetName <String> -VirtualNetworkName <String> [-IgnoreMissingServiceEndpoint] [-PassThru]

```

### SubnetIdParameterSet
```
Add-AzWebAppAccessRestrictionRule [-ResourceGroupName] <String> [-WebAppName] <String> [-Name <String>]
 [-Description <String>] -Priority <UInt32> [-Action <String>] [-SlotName <String>] [-TargetScmSite]
 -SubnetId <String> [-IgnoreMissingServiceEndpoint] [-PassThru] [-DefaultProfile <IAzureContextContainer>]

```

## 설명은
**Add-AzWebAppAccessRestrictionRule** Cmdlet은 Azure 웹 앱에 액세스 제한 규칙을 추가 합니다.

## 예제의

### 예제 1: 웹 앱에 IpAddress 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name IpRule -Priority 200 -Action Allow -IpAddress 10.10.0.0/8
```

이 명령은 리소스 그룹 Default-웹용 WestUS에 속한 ContosoSite 이라는 웹 앱에 우선 순위 200 및 ip 범위에 대 한 액세스 제한 규칙을 추가 합니다.

### 예제 2: 웹 앱에 서브넷 서비스 끝점 액세스 제한 규칙 추가
```
PS C:\>Add-AzWebAppAccessRestrictionRule -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-Name SubnetRule -Priority 300 -Action Allow -SubnetName appgw-subnet -VirtualNetworkName corp-vnet
```

이 명령은 appgw에 우선 순위 300를 사용 하는 액세스 제한 규칙을 추가 하 고, corp-vnet의 서브넷-subnet에는 리소스 그룹 기본값-웹 WestUS에 속하는 ContosoSite 라는 웹 앱에 연결 합니다.
