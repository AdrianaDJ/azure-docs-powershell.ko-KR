# Get-AzureRemoteAppLocation

## SYNOPSIS
Azure RemoteApp에서 지 원하는 위치를 나열 합니다.

## 구문과

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 설명은
**AzureRemoteAppLocation** Cmdlet은 Azure RemoteApp에서 지 원하는 위치를 나열 합니다.
