# Get-AzNetworkServiceTag

## SYNOPSIS
서비스 태그 정보 리소스의 목록을 가져옵니다.

## 구문과

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**Get-AzNetworkServiceTag** cmdlet은 서비스 태그 정보 리소스 목록을 가져옵니다.

지정 하는 Azure 지역 정보는 위치를 기반으로 하는 필터가 아니라 버전에 대 한 참조로 사용 됩니다. 예를 들어 사용자가 지정 하는 경우에도 `-Location eastus2` 모든 지역에서 접두사 세부 정보를 사용 하 여 서비스 태그 목록을 가져오고 구독이 속해 있는 클라우드로 제한 (예: 공개, 미국 정부, 중국 또는 독일) 할 수 있습니다.

두 번째 명령은 목록에서 Sql에 대 한 정보 리소스를 선택 하도록 필터링 합니다.
