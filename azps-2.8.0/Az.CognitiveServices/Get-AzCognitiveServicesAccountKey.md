# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
계정에 대 한 API 키를 가져옵니다.

## 구문과

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzCognitiveServicesAccountKey** cmdlet은 프로 비전 된 인식 서비스 계정에 대 한 API 키를 가져옵니다.
인지 서비스 계정에는 Key1 및 Key2의 두 가지 API 키가 있습니다.
키를 사용 하면 인식 서비스 계정 끝점과 상호 작용할 수 있습니다.
New-AzCognitiveServicesAccountKey를 사용 하 여 키를 다시 생성 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
