# Stop-AzureRmAutomationJob

## SYNOPSIS
자동화 작업을 중지 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzureRmAutomationJob** Cmdlet은 Azure Automation 작업을 중지 합니다.
실행 중인 자동화 작업을 지정 합니다.

## 예제의

### 예제 1: 작업 중지
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

이 명령은 지정 된 ID를 가진 작업을 중지 합니다.
