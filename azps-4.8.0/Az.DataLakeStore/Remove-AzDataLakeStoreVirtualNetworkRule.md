# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
지정 된 Data Lake Store에서 지정 된 가상 네트워크 규칙을 제거 합니다.

## 구문과

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzDataLakeStoreVirtualNetworkRule** cmdlet은 지정 된 Data Lake store에서 지정 된 가상 네트워크 규칙을 제거 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

가상 네트워크 규칙 "myVNET"에서 계정 "dl"을 제거 합니다.
