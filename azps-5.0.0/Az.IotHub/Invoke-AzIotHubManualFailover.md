# Invoke-AzIotHubManualFailover

## SYNOPSIS
IoT Hub에 대 한 장애 조치 프로세스를 geo 페어링 재해 복구 지역으로 호출 합니다.

## 구문과

### ResourceSet (기본값)
```
Invoke-AzIotHubManualFailover [-ResourceGroupName] <String> [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubManualFailover [-InputObject] <PSIotHub> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubManualFailover [-ResourceId] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
이는 IoT hub의 장애 조치를 보조 위치로 트리거합니다. 이 작업을 수행 하면 솔루션에 시간 및 원격 분석 손실이 발생 합니다. 이 작업은 장기 실행 이며 완료 하는 데 몇 분 정도 걸릴 수 있습니다. 사용 하는 경우 주의 해 주십시오.

## 예제의

### 예제 1
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

"Myiothub" IoT Hub의 장애 조치 프로세스를 시작 합니다.

### 예제 2
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub" -AsJob
```

백그라운드에서 "myiothub" IoT Hub의 장애 조치 프로세스를 시작 합니다.
