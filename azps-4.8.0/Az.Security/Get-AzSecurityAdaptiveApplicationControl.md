# Get-AzSecurityAdaptiveApplicationControl

## SYNOPSIS
구독에 대 한 응용 프로그램 컨트롤 VM/서버 그룹의 목록을 가져옵니다.

## 구문과

### 구독 범위 (기본값)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
적응 응용 프로그램 컨트롤은 Azure 보안 센터에서 자동으로 계산 되며,이 cmdlet을 사용 하 여 구독 범위에서 적응 응용 프로그램 컨트롤 리소스의 목록을 가져옵니다.
