# Reset-AzAttestationPolicy

## SYNOPSIS
Azure Attestationn의 테 넌 트에서 정책을 다시 설정 합니다.}

## 설명은
Reset-AzAttestationPolicy cmdlet은 Azure 증명의 테 넌 트에서 사용자가 정의한 증명 정책을 다시 설정 합니다.

## 예제의

### 예제 1
```powershell
PS C:\> Reset-AzAttestationPolicy -Name pshtest -ResourceGroupName psh-test-rg -Tee SgxEnclave
```

T e t type *SgxEnclave* 에 대 한 증명 공급자 *pshtest* 의 기본값으로 정책을 다시 설정 합니다.

### 예제 2
```powershell
PS C:\> $resetJwt = Get-Content -Path .\reset.policy.txt.signed.txt
PS C:\> Reset-AzAttestationPolicy -Name pshtest -ResourceGroupName psh-test-rg -Tee SgxEnclave -Policy $resetJwt
```

증명 공급자 *pshtest* 가 격리 된 신뢰 모델을 사용 하도록 구성 된 경우에는 서명 된 정책을 포함 하 여 *SgxEnclave* 에서 정책을 t e 티 형식에 대 한 기본값으로 다시 설정 합니다.
