# Stop-AzureBatchJob

## SYNOPSIS
일괄 작업을 중지 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Stop-AzureBatchJob [-Id] <String> [[-TerminateReason] <String>] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**Stop-AzureBatchJob** Cmdlet은 Azure 일괄 처리 작업을 중지 합니다.
이 명령은 작업을 완료 된 것으로 표시 합니다.

## 예제의

### 예제 1: 일괄 작업 중지
```
PS C:\>Stop-AzureBatchJob -Id "Job-000001" -TerminateReason "No more tasks to run" -BatchContext $Context
```

이 명령은 ID 작업이 있는 작업을 중지 합니다-000001.
명령은 작업을 중지 하도록 선택한 이유를 지정 합니다.
Get-AzureRmBatchAccountKeys cmdlet을 사용 하 여 컨텍스트를 $Context 변수에 할당 합니다.
