# Remove-AzureRmSiteRecoveryVault

## SYNOPSIS
사이트 복구 자격 증명 모음을 제거 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Remove-AzureRmSiteRecoveryVault -Vault <ASRVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
**AzureRmSiteRecoveryVault** Cmdlet은 Azure Site Recovery 자격 증명 모음을 삭제 합니다.
