# Get-AzMarketplacePrivateStore

## SYNOPSIS
개인 스토어 목록 가져오기

## 구문과

```
Get-AzMarketplacePrivateStore [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
테 넌 트 범위 아래에 만들어진 개인 저장소 목록 가져오기

## 예제의

### 예제 1
```powershell
PS C:\> Get-AzMarketplacePrivateStore
```
