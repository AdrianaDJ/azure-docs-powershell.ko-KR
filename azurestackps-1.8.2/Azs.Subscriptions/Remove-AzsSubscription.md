# Remove-AzsSubscription

## SYNOPSIS
지정한 구독을 삭제 합니다.

## 구문과

```
Remove-AzsSubscription [-SubscriptionId] <String> [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
지정한 구독을 삭제 합니다.

## 예제의

### 예제 1
```
Remove-AzsSubscription -SubscriptionId d387f779-85d8-40b6-8607-8306295ebff9
```

지정한 구독을 삭제 합니다.
