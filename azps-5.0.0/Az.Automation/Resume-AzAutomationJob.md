# Resume-AzAutomationJob

## SYNOPSIS
일시 중단 된 자동화 작업을 다시 시작 합니다.

## 구문과

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzAutomationJob** cmdlet은 일시 중단 된 Azure 자동화 작업을 다시 시작 합니다.
일시 중단 된 작업을 지정 합니다.
작업을 일시 중단 하려면 Suspend-AzAutomationJob cmdlet을 사용 합니다.

## 예제의

### 예제 1: 일시 중단 된 작업 다시 시작
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

이 명령은 지정 된 ID를 가진 작업을 다시 시작 합니다.
