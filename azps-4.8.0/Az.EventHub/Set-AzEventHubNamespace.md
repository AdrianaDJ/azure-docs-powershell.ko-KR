# Set-AzEventHubNamespace

## SYNOPSIS
지정 된 이벤트 허브 네임 스페이스를 업데이트 합니다.

## 설명은
Set-AzEventHubNamespace cmdlet은 지정 된 이벤트 허브 네임 스페이스의 속성을 업데이트 합니다.
