# New-AzsEncryptionKeyBase64

## SYNOPSIS

## 구문과

```
New-AzsEncryptionKeyBase64 [<CommonParameters>]
```

## 설명은
인프라 백업용 암호화 키를 생성 합니다.

## 예제의

### 예제 1
```

```
