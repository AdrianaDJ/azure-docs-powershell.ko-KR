# Get-AzureRmSqlDatabaseAdvisor

## SYNOPSIS
Azure SQL 데이터베이스에 대 한 하나 이상의 관리자를 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 설명은
**AzureRmSqlDatabaseAdvisor** Cmdlet은 Azure sql database에 대 한 하나 이상의 Azure sql 데이터베이스 관리자를 가져옵니다.
