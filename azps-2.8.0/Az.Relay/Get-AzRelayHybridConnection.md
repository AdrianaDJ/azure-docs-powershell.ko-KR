# Get-AzRelayHybridConnection

## SYNOPSIS
릴레이 네임 스페이스 내에서 지정 된 HybridConnection에 대 한 설명을 가져옵니다.

## 구문과

```
Get-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
**AzRelayHybridConnection** Cmdlet은 Relay 네임 스페이스 내에서 지정 된 HybridConnection에 대 한 설명을 가져옵니다.
