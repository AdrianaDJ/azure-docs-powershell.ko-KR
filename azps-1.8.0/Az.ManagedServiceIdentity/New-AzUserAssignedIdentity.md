# New-AzUserAssignedIdentity

## SYNOPSIS
새 사용자에 게 할당 된 Id를 만들거나 기존 사용자에 게 할당 된 Id를 업데이트 합니다.

## 구문과

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzUserAssignedIdentity** cmdlet은 새 사용자 지정 id를 만듭니다. 이미 기존 id와 함께 사용 하면 id를 업데이트 합니다.
Azure 리소스 관리자 태그를 id에 추가 하려면 Set-AzResource cmdlet을 사용 하세요.
