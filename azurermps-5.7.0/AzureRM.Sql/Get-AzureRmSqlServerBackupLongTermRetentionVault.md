# Get-AzureRmSqlServerBackupLongTermRetentionVault

## SYNOPSIS
서버 장기 보존 자격 증명 모음을 가져옵니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Get-AzureRmSqlServerBackupLongTermRetentionVault [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmSqlServerBackupLongTermRetentionVault** cmdlet은이 서버에 등록 된 장기 보존 자격 증명을 가져옵니다.
자격 증명 모음은 백업 데이터를 저장 하는 데 사용 되는 Azure Backup 리소스입니다.
