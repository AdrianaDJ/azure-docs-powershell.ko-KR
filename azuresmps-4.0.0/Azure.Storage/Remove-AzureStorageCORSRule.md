# Remove-AzureStorageCORSRule

## SYNOPSIS
저장소 서비스의 CORS를 제거 합니다.

## 구문과

```
Remove-AzureStorageCORSRule [-ServiceType] <StorageServiceType> [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>]
 [<CommonParameters>]
```

## 설명은
**AzureStorageCORSRule** Cmdlet은 Azure 저장소 서비스의 CORS (교차 원본 리소스 공유)를 제거 합니다.
이 cmdlet은 저장소 서비스 유형에 서 모든 CORS 규칙을 삭제 합니다.
이 cmdlet에 대 한 저장소 서비스의 유형은 Blob, 테이블, 큐 및 파일입니다.

## 예제의

### 예제 1: blob 서비스에 대 한 CORS 규칙 제거
```
PS C:\>Remove-AzureStorageCORSRule -ServiceType Blob
```

이 명령은 Blob 서비스 형식에 대 한 CORS 규칙을 제거 합니다.
