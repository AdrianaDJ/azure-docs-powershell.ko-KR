# Get-AzsStorageSettings

## SYNOPSIS
저장소 리소스 공급자 설정을 반환 합니다.

## 구문과

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## 설명은
저장소 리소스 공급자 설정을 반환 합니다.

## 예제의

### 예제 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

저장소 설정을 가져옵니다.
