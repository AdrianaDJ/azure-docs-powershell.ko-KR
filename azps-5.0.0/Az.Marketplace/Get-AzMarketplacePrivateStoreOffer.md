# Get-AzMarketplacePrivateStoreOffer

## SYNOPSIS
하나 이상의 개인 저장소 제안을 가져옵니다.

## 구문과

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
테 넌 트 범위 아래에 추가 된 공개 + 전용 요금제를 사용 하 여 하나 이상의 개인 저장소 서비스를 가져옵니다. 구독 id가 표시 되는 경우 구독 범위 아래에 있는 비공개 요금제를 사용 하 여 하나 이상의 개인 저장소 제안을 가져옵니다.
