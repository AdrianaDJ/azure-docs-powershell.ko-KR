# Remove-AzureRmDataLakeStoreAccount

## SYNOPSIS
Data Lake Store 계정을 영구적으로 삭제 합니다.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 구문과

```
Remove-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzureRmDataLakeStoreAccount** Cmdlet은 Data Lake store 계정을 영구적으로 삭제 합니다.

## 예제의

### 예제 1: Data Lake Store 계정 제거
```
PS C:\>Remove-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

이 명령은 Data Lake Store에서 ContosoADL 이라는 계정을 제거 합니다.
