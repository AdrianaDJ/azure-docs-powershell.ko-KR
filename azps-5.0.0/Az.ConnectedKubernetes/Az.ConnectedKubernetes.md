# Az ConnectedKubernetes 모듈
## 설명은
Microsoft Azure PowerShell: ConnectedKubernetes cmdlet

## Az ConnectedKubernetes Cmdlet
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
이름, id, 속성, 추가 클러스터 세부 정보를 포함 하 여, 지정 된 연결 클러스터의 속성을 반환 합니다.

### [새로운 AzConnectedKubernetes](New-AzConnectedKubernetes.md)
새 K8s 클러스터를 등록 하 여 ARM에서 추적 된 리소스를 만드는 API

### [제거-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
연결 된 클러스터를 삭제 하 고 (ARM (Azure Resource Manager)에서 추적 된 리소스를 제거 합니다.

### [업데이트-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
연결 된 클러스터 리소스의 특정 속성을 업데이트 하는 API
