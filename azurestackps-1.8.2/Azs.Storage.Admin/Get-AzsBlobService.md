# Get-AzsBlobService

## SYNOPSIS
Blob 서비스를 반환 합니다.

## 구문과

```
Get-AzsBlobService [-FarmName] <String> [-ResourceGroupName <String>] [<CommonParameters>]
```

## 설명은
Blob 서비스를 반환 합니다.

## 예제의

### 예제 1
```
Get-AzsBlobService -FarmName f9b8e2e2-e4b4-44e0-9d92-6a848b1a5376
```

Blob 서비스를 가져옵니다.
