# Remove-AzureDns

## SYNOPSIS
Azure 서비스에서 DNS 서버를 제거 합니다.

## 구문과

```
Remove-AzureDns [-Name] <String> [-ServiceName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## 설명은
**제거-AzureDns** Cmdlet은 Azure 서비스에서 DNS 서버를 제거 합니다.

## 예제의

### 예제 1: 서비스에서 DNS 서버 제거
```
PS C:\> Remove-AzureDns -ServiceName "ContosoService" -Name "Dns07"
```

이 명령은 Dns07 이라는 DNS 서버를 ContosoService에서 제거 합니다.
