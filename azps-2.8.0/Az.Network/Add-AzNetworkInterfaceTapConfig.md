# Add-AzNetworkInterfaceTapConfig

## SYNOPSIS
NetworkInterface와 연결 된 TapConfiguration 리소스를 만듭니다. 기존 VirtualNetworkTap 리소스를 참조 합니다.

## 구문과

### SetByResource (기본값)
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTap <PSVirtualNetworkTap>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceId
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTapId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 설명은
**AzNetworkInterfaceTapConfig** Cmdlet은 NetworkInterface와 연결 된 TapConfiguration 리소스를 만듭니다. 기존 VirtualNetworkTap 리소스를 참조 합니다.

## 예제의

### 예제 1: 지정 된 NetworkInterface에 TapConfiguration 추가
```
PS C:\>Add-AzNetworkInterfaceTapConfig -NetworkInterface $sourceNic -VirtualNetworkTap $vVirtualNetworkTap -Name 'myTapConfig'
```

TapConfiguration를 sourceNic에 추가 합니다. SourceNic VM의 트래픽이 vVirtualNetworkTap 리소스에서 참조 하는 대상 VM으로 미러링 됩니다.
