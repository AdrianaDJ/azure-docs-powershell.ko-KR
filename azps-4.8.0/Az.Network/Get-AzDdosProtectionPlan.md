# Get-AzDdosProtectionPlan

## SYNOPSIS
DDoS protection 계획을 가져옵니다.

## 구문과

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 설명은
Get-AzDdosProtectionPlan cmdlet은 DDoS protection 계획을 가져옵니다.

## 예제의

### 예제 1: 특정 DDoS 보호 계획 가져오기

```
D:\> Get-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName

                      {

```

이 경우에는 각각 리소스 그룹과 DDoS 보호 계획의 이름에 해당 하는 **ResourceGroupName** 및 **Name** 특성을 모두 지정 해야 합니다.
