# Remove-AzSqlServer

## SYNOPSIS
Azure SQL 데이터베이스 서버를 제거 합니다.

## 구문과

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 설명은
**AzSqlServer** Cmdlet은 Azure SQL 데이터베이스 서버를 제거 합니다.
삭제 작업은 비동기적 이며 시간이 걸릴 수 있으므로 완전히 삭제 되는 서버에 의존 하는 추가 작업을 수행 하기 전에 작업이 완료 되었는지 확인 합니다.
예를 들어 동일한 이름을 사용 하는 새 서버를 만들어야 합니다.

## 예제의

### 예제 1: 서버 제거
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

이 명령은 Server01 이라는 Azure SQL 데이터베이스 서버를 제거 합니다.
